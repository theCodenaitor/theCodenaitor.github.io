## 🚀 How to Use Evergreen Theme URLs

### Step 1: Find Your Theme CSS URL 🔗

First, identify the CSS file URL of the theme you want to use. For example:
```
https://thecodenaitor.github.io/hebit-css/evergreen.css
```

### Step 2: Add the URL to Evergreen 📝

1. Visit the site (or the Evergreen customization box)
2. Locate the **"Site URL of Evergreen"** input box
3. Paste your CSS theme URL into the box
4. Apply the theme and enjoy! ✨

### Example Theme URL:
```
https://thecodenaitor.github.io/hebit-css/evergreen.css
```

## 💡 Important: Cache Busting with Version Parameters

Sometimes, browsers cache CSS files, and your changes might not appear immediately. To force a refresh, **add a version parameter** (`?v=`) to your URL:

```
https://thecodenaitor.github.io/hebit-css/evergreen.css?v=1
https://thecodenaitor.github.io/hebit-css/evergreen.css?v=2
https://thecodenaitor.github.io/hebit-css/evergreen.css?v=3
```

**How it works:**
- Each time you change the version number (1, 2, 3, etc.), the browser treats it as a new URL
- This forces a fresh download of the CSS file, bypassing the cache
- Increment the number whenever you want to see the latest changes 🔄

### ✅ Tips:
- Use timestamps or dates: `?v=20260505` (YYYYMMDD format)
- Use incremental numbers: `?v=1`, `?v=2`, `?v=3`
- Add the parameter after the CSS filename

## ❓ Troubleshooting

### Theme not appearing?
✔️ Check the URL is correct  
✔️ Add `?v=1` to bypass cache  
✔️ Hard refresh your browser (Ctrl+Shift+R or Cmd+Shift+R)

### Changes not showing?
✔️ Increment the version number: `?v=1` → `?v=2`  
✔️ Wait a moment for the browser to fetch the new file
